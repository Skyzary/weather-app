# 🌤️ Погодное приложение (Weather App)

Это современное погодное приложение, которое предоставляет информацию о погоде в реальном времени для любого города. Приложение создано с акцентом на чистый, интуитивно понятный пользовательский интерфейс и надежную масштабируемую архитектуру.

## ✨ Особенности

* **Текущая погода:** Получайте актуальную информацию о температуре, влажности, скорости ветра и погодных условиях для любого города.
* **Динамические фоны:** Красивые изображения, синхронизированные с погодой благодаря Unsplash API.
* **Поиск:** Удобный поиск любого города мира.
* **Адаптивный дизайн:** Приложение полностью адаптивно и корректно отображается на всех устройствах.
* **Современный UI:** Красивый интерфейс с эффектами свечения и круговыми индикаторами прогресса.
* **Интернационализация:** Поддержка английского, французского, русского и украинского языков с автоматическим определением по геолокации.

## ⚙️ Начало работы

Чтобы запустить проект локально, выполните следующие шаги.

### 1. Требования

* **Node.js** (v18 или выше)
* **npm**

### 2. Настройка API

Для полноценной работы приложения требуются два API ключа:

* **OpenWeatherMap API** (Данные о погоде):
    1. Перейдите на [OpenWeatherMap](https://openweathermap.org/api) и создайте аккаунт.
    2. Создайте новый ключ в панели управления (активация может занять до 2 часов).
* **Unsplash API** (Фоновые изображения):
    1. Перейдите на [портал для разработчиков Unsplash](https://unsplash.com/developers).
    2. Зарегистрируйтесь как разработчик и создайте **New Application**.
    3. Скопируйте ваш **Access Key** из панели управления.

### 3. Установка

1. **Клонируйте репозиторий**
    ```sh
    git clone https://github.com/your_username/weather-app.git
    cd weather-app
    ```

2. **Настройте переменные окружения** Создайте файл `.env` в корневой директории и добавьте ваши ключи:
    ```env
    VITE_API_KEY=ваш_ключ_weather_api
    VITE_UNSPLASH_ACCESS_KEY=ваш_ключ_unsplash_api
    ```

3. **Установите NPM зависимости**
    ```sh
    npm install
    ```

4. **Запустите сервер для разработки**
    ```sh
    npm run dev
    ```

В директории проекта можно запустить:

* `npm run build`: Сборка приложения для продакшена.
* `npm run lint`: Проверка кода линтером.
