# 🌤️ Додаток Погоди (Weather App)

Це сучасний додаток для перегляду погоди, який надає інформацію в реальному часі для будь-якого міста. Додаток створено з фокусом на чистий, інтуїтивно зрозумілий інтерфейс користувача та надійну, масштабовану архітектуру.

## ✨ Особливості

* **Поточна погода:** Отримуйте актуальну температуру, вологість, швидкість вітру та погодні умови для будь-якого міста.
* **Динамічні фони:** Красиві зображення, синхронізовані з погодою завдяки Unsplash API.
* **Пошук:** Зручний пошук будь-якого міста світу.
* **Адаптивний дизайн:** Додаток повністю адаптивний і коректно працює на всіх пристроях.
* **Сучасний UI:** Красивий інтерфейс з ефектами світіння та круговими індикаторами прогресу.
* **Інтернаціоналізація:** Підтримка англійської, французької, російської та української мов з автоматичним визначенням по геолокації.

## ⚙️ Початок роботи

Щоб запустити проект локально, виконайте наступні кроки.

### 1. Вимоги

* **Node.js** (v18 або вище)
* **npm**

### 2. Налаштування API

Для повноцінної роботи додатка потрібні два API ключі:

* **OpenWeatherMap API** (Дані про погоду):
    1. Перейдіть на [OpenWeatherMap](https://openweathermap.org/api) та створіть акаунт.
    2. Створіть новий ключ у панелі керування (активація може зайняти до 2 годин).
* **Unsplash API** (Фонові зображення):
    1. Перейдіть на [портал для розробників Unsplash](https://unsplash.com/developers).
    2. Зареєструйтеся як розробник і створіть **New Application**.
    3. Скопіюйте ваш **Access Key** з панелі керування.

### 3. Встановлення

1. **Клонуйте репозиторій**
    ```sh
    git clone https://github.com/your_username/weather-app.git
    cd weather-app
    ```

2. **Налаштуйте змінні оточення** Створіть файл `.env` у кореневій директорії та додайте ваші ключі:
    ```env
    VITE_API_KEY=ваш_ключ_weather_api
    VITE_UNSPLASH_ACCESS_KEY=ваш_ключ_unsplash_api
    ```

3. **Встановіть NPM залежності**
    ```sh
    npm install
    ```

4. **Запустіть сервер для розробки**
    ```sh
    npm run dev
    ```

## 📜 Доступні скрипти

У директорії проекту можна запустити:

* `npm run dev`: Запуск додатка в режимі розробки.
* `npm run build`: Збірка додатка для продакшена.
* `npm run lint`: Перевірка коду лінтером.
* `npm run preview`: Локальний запуск зібраного додатка.
